# Karabiner-Multi-Layered

Multi-Layered Keymap for Japanese Keyboards: Cursor, Mouse, NumPad, Web Browser, etc. for [Karabiner-Elements](https://karabiner-elements.pqrs.org/).

## Build

```shell
deno task json
```

### Development

```shell
deno task format
deno task lint
deno task dev
```
